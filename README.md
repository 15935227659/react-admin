# React-base

React脚手架，专用作后台管理系统，并且提供丰富的示例代码

## 技术选型

* classnames + lodash
* redux-action + redux-logger
* react + redux + react-router + Immutable + Fetch + ant-design
* webpack + hot-loader

## 使用方法

```bash
$ git clone https://github.com/bodyno/react-starter-kit.git
$ cd react-base
$ cnpm install                   # 需要安装cnpm
$ npm start                     
```
